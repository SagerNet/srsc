# Endpoint

### Structure

```json
{
  "endpoints": {
    "<endpoint_path>": {
      "type": ""
    }
  }
}
```

### Fields

#### endpoint_path

==Required==

Endpoint path of the HTTP server.

#### type

==Required==

| Type   | Format          | 
|--------|-----------------|
| `file` | [File](./file/) |
